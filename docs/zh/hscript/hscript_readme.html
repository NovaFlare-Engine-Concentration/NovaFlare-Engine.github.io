看我

注意事项

  • 在这个Hscript中因为一些不可抗因素会导致一个问题，你往Array里存一个自定义类的对象，并且这个对象继承了某个类，那么这个对象会被强制转换成你所继承的类（父类），导致你不能使用这个对象里面的自定义函数和变量。

如果你有一个这样的自定义类

HScript 

package some;

class YouObject extends FlxSprite
{
    public function new()
    {
        super();
    }
    public function knock()
    {
        trace("who's that knocking ?");
    }
}

错误示例

HScript 

package blabla;

import some.YouObject;

class YouNextClass
{
    var oneArray:Array = [];
    public function new()
    {
        var Obj = new YouObject();
        Obj.knock();  // who's that knocking ?
        
        oneArray.push(Obj);
        oneArray.[0].knock();   // Error: Function not found 'knock'

        /*
        因为在你直接使用Array的索引作为对象并直接调用这个对象的函数时，
        这个YouObject因为某些原因被转换成了他的父类FlxSprite，
        因为父类里本来就没有这个function knock()
        所以他就报错了
        */
    }
}

注意事项

  • 如果你不想要让他强制把子类转成父类那么就把以下元数据添加到你往Array存对象的位置。

正确示例

HScript 

package blabla;

import some.YouObject;

@:unpass_standard  //放到你需要的地方。

class YouNextClass
{
    var oneArray:Array = [];
    public function new()
    {
        var Obj = new YouObject();
        Obj.knock();  // who's that knocking ?
        
        oneArray.push(Obj);
        oneArray.[0].knock();   // who's that knocking ?
    }
}
