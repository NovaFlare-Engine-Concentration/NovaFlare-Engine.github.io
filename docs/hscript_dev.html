核心函数参考

函数: createSprite()

createSprite(x:Float, y:Float, ?graphic:GraphicAsset):Sprite

在指定位置创建一个新的精灵对象，可选择性地指定图形资源。

这是NovaFlare Engine中创建视觉元素的主要方法。它返回一个Sprite对象，可以添加到显示列表并通过各种属性和方法进行操作。

注意事项

  • 精灵在添加到显示对象容器之前不会可见。
  • 如果没有提供图形资源，精灵将创建为空容器。
  • 坐标相对于父容器的坐标系。

使用示例

Haxe 
// 创建一个位于屏幕中央的精灵
var playerSprite:Sprite = createSprite(400, 300, "assets/images/player.png");

// 设置精灵属性
playerSprite.scaleX = playerSprite.scaleY = 0.5;
playerSprite.alpha = 0.8;

// 将精灵添加到舞台
addChild(playerSprite);

// 为精灵添加动画
playerSprite.animation.add("walk", [0, 1, 2, 3], 10, true);
playerSprite.animation.play("walk");

函数: playSound()

playSound(soundId:String, ?volume:Float = 1.0, ?looped:Bool = false):SoundChannel

播放音效或音乐轨道，可指定参数。

注意事项

  • 声音必须使用资源加载系统预先加载。
  • 音量范围是0.0（静音）到1.0（最大音量）。

使用示例

Haxe 
// 播放跳跃音效
var jumpSound:SoundChannel = playSound("jump", 0.7);

// 播放背景音乐（循环）
var bgMusic:SoundChannel = playSound("bgm_main", 0.5, true);

// 停止背景音乐
bgMusic.stop();
